УСТАНОВКА
-----------------
Стандартный процесс установки, общий для всех модулей Prestashop, применяется к 
«Генератору Cкидок». Если Вы используете «Мультимагазин», перейдите в «Конфигурацию» модуля (кнопка «Настроить») и установите флажок «Активировать модуль для этого контекста магазина: all shops». Это позволит генерировать скидки для любого из магазинов по отдельности или для всех сразу. Если Вам нужно прекратить использование модуля на какое-то время, используйте кнопку «Отключить».

Важно: Если по какой-либо причине вы хотите полностью удалить модуль «Генератор Cкидок», сначала ДЕИНСТАЛЛИРУЙТЕ его, а затем УДАЛИТЕ. Этот порядок действий важен для очистки таблиц базы данных, связанных с модулем.

Важно: Поскольку модуль «Генератор Скидок» расширяет встроенные функции правил корзины, необходимо отключить опции «Отключить все переопределения» и «Отключить не PrestaShop модули» в разделе «Расширенные параметры» > «Производительность» > «Режим отладки».

ОПИСАНИЕ
-----------------
Модуль «Генератор Скидок» позволяет генерировать множество скидок (правил корзины) с уникальными промо-кодами. Вы можете сгенерировать скидки для определенного продукта, группы выбранных продуктов, определенной категории или всего заказа или указать любые другие условия, доступные для стандартных правил корзины.

Вы сможете определить:
- сколько скидок с уникальными кодами вы хотите сгенерировать
- структуру уникальных кодов (комбинацию цифр и букв)
- сколько раз скидка может использоваться клиентом и т.д.

НАСТРОЙКИ МОДУЛЯ
-----------------
Этот модуль устанавливается непосредственно на вкладку Каталог > Скидки и расширяет базовые функции.

1. В Prestashop 1.7 перейдите на вкладку «Каталог» > «Скидки» и нажмите «Добавить новое правило для корзины». В Prestashop 1.5 - 1.6 перейдите на вкладку меню «Ценовые правила» > «Правила корзины» и нажмите «Добавить новое правило для корзины».
2. Чтобы создать скидки с помощью «Генератора Cкидок», установите флажок «Создать много уникальных скидок» и заполните новые поля, которые появятся на странице.
3. Заполните новые обязательные поля:
    - Общее количество уникальных скидок: количество скидок для генерации.
    - Конфигурация кода:
        - Префикс: разрешены любые буквы или цифры. Следующие символы недопустимы: ^!,;? = + () @ "° {} _ $%. Префикс - это стабильная часть Вашего кода, он общий для всех скидок.
        - Маска: последовательность из X и/или Y, которая определяет структуру кода. X обозначает любое число, Y - любую букву латинского алфавита. X и Y будут генерироваться случайным образом, чтобы сделать ваши коды уникальными. Пример: если Prefix = TEST- и Mask = XXYY, модуль сгенерирует такие коды, как TEST-96FA, TEST-27ME и т. д.
4. Убедитесь, что все остальные обязательные поля заполнены правильно.
5. Сохраните форму. Модуль сгенерирует количество скидок, которые вы указали.

ЭКСПОРТ СПИСКОВ
-----------------
Все сгенерированные скидки будут отображены в таблице «История модуля», расположенной внизу страницы конфигурации модуля.

Существует три типа списков, загружаемых в виде файлов CSV:

- «Все» - отобразит все сгенерированные скидки, дату начала и окончания, тип скидки. Этот список генерируется в момент создания скидки и не меняется.
- «Использованные» - отобразит только те скидки, которые были использованы клиентами, с указанием имени и email. Этот список динамический и обновляется  при скачивании.
- «Неиспользованные» - отобразит только те скидки, которые еще не были использованы. Этот список динамический и обновляется при скачивании.
