**HOW TO INSTALL THE MODULE**

1. Access the section "_Modules -> Modules and services_".
2. Add the file "_.zip_".
3. When you see the module "_Related Products Pro_", click on the install button.
4. If you access to the configuration screen, you will see two tabs, one to create the related product blocks and the other to view and modify the block settings.

**WHAT APPORTS**

With this module you can create blocks of related products in the product page based on the value of a feature, an attribute or a reference. You can also modify different configuration parameters such as if you want to see the products in slider format or in grid format. It also incorporates the lazyload technique to improve page performance.

RELATED PRODUCT BLOCK CREATION TAB

1. Write the product ID that you want to add a block of related products. Or even better! Start by writting the name of the product and select it with the dropdown that will be generated with the auto-complete function.
2. Select if you want to create a block based on a feature, an attribute or a reference.
3. If you have selected a feature, select the one you want, wait for its values ​​to be filled in and select the one you want.
4. If you have selected an attribute, select the one you want, wait for its values ​​to be filled in and select the one you want.
5. If you have selected a reference, enter the value you want the reference of the related products to contain.
6. Click Save.

CONFIGURATION TAB

You can indicate different settings such as:
- If you want to show the product block as a slider or as a grid.
- If you want to show a title or not.
- If you want to load the images of the products with the lazyload technique to improve the performance of the page.
- Customize the title of the blocks.
